# Stardew Valley Calendar

Локальный русскоязычный фан-помощник по Stardew Valley. Эта версия пересобрана как набор игровых предметных интерфейсов: рабочий стол фермера, календарная доска, комнаты Центра, журналы, слоты, чеклисты и фильтры по стадии прохождения.

## Что внутри

- Главный экран без обычного сайдбара: верхняя игровая рамка, предметные вкладки, календарь сезона, дневная доска и быстрый маршрут.
- Выбор сезона, дня, погоды и стадии: `Год 1`, `Год 2+`, `Пустыня`, `Остров`, `Перфекшен`.
- `Центр`: интерактивные комнаты, прогресс, предметные слоты, подсказки источников и нормализованные награды вместо сырых id.
- `Рыбалка`: обычная рыба, легендарная, ловушки и особая рыба разнесены по внутренним закладкам.
- `Ферма`: посевы, сбор, прибыль и теплица живут в одном разделе, но визуально и логически разделены.
- Главный экран показывает последний шанс по посадкам и сколько контента скрыто текущей стадией прохождения.
- Жители с портретами, днями рождения, любимыми подарками и недельной отметкой.
- Жители используют локальные `Characters/schedules/*.json`: выбранный день показывает подходящий маршрут, включая дождь, сезон и день недели.
- Каталог предметов как мини-вики с панелью деталей, связями с узелками, рецептами, крафтом и подарками.
- Готовка и крафт с ингредиентами и источниками рецептов.
- `Endgame`: Имбирный остров, лодка Вилли, орехи, вулкан, Qi и поздняя прогрессия.
- `Perfection`: отдельный ledger 100% целей.
- Экспорт и импорт локальных отметок через JSON в нижней панели.

## Запуск

```powershell
npm install
npm run generate
npm run dev -- --port 5174
```

Открыть: http://127.0.0.1:5174

## Проверки

```powershell
npm run typecheck
npm run lint
npm run build
npm run smoke
```

`npm run smoke` делает скриншоты главного экрана, календаря, Центра, рыбалки, фермы, жителей, каталога, рецептов, Endgame, Perfection и мобильного вида в `work/`; дополнительно проверяет консольные ошибки, битые картинки и явные переполнения блоков.
